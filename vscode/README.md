# HULK for VS Code

Syntax highlighting and support for the HULK programming language.

## Features

- **Syntax Highlighting**: Support for the full HULK specification, including:
    - Keywords, Operators, and Literals.
    - Protocols and Inheritance.
    - Macros with symbolic arguments (`@`), placeholders (`$`), and variable arguments (`*`).
- **Snippets**: Quick access to common HULK constructs like `function`, `type`, `let`, `if`, and `def`.
- **Language Configuration**: Bracket matching, comment toggling, and code folding.

## Extension Settings

This extension contributes the following settings:

* `hulk.enable`: Enable/disable this extension.
